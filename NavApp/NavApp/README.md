# Práctica: Navegación entre Pantallas
**POO II – Desarrollo de aplicaciones móviles en Android**

Aplicación Android con navegación funcional entre 3 pantallas usando Jetpack Compose y Navigation Component.

## Pantallas

- **Inicio** – punto de entrada, botones para navegar a las otras pantallas
- **Detalle** – contador interactivo con botones + y −
- **Perfil** – switch de notificaciones y botón para volver al inicio

## Cómo ejecutar

1. Clonar el repositorio
2. Abrir en Android Studio
3. Conectar dispositivo o emulador (API 24+)
4. Presionar Run ▶
